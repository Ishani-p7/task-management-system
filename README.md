TaskFlow is a full-stack task manager using Node.js + TypeScript (backend) and Next.js (frontend).

Key Features
JWT auth (access + refresh tokens)
Task CRUD, filters, search, pagination
Dashboard with stats
Auto token refresh & persistent login
Responsive UI with modals, toasts, and validation
Tech Stack
Backend: Node.js, Express, Prisma
DB: SQLite / PostgreSQL
Frontend: Next.js, Tailwind, React Hook Form
Auth & Security: JWT, bcrypt, rate limiting, validation
Core APIs
Auth: register, login, refresh, logout, me
Tasks: create, read, update, delete, stats, toggle status
Highlights
Secure token rotation
Ownership-based access control
Clean modular architecture
